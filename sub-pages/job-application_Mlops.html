Responsibilities

  • Develop and maintain robust, scalable, and efficient MLOps pipelines for deploying and managing machine learning models.
  • Collaborate with data scientists, AI/ML engineers, and software developers to integrate machine learning models into production environments.
  • Automate the deployment, monitoring, and maintenance of ML models to ensure high availability and performance.
  • Implement CI/CD pipelines to streamline model training, testing, and deployment processes.
  • Ensure the scalability and reliability of ML systems through effective monitoring and logging.
  • Troubleshoot and resolve issues related to ML model performance and deployment.
  • Work with cloud platforms (AWS, Google Cloud, Azure) to manage infrastructure for ML workloads.
  • Stay updated with the latest trends and advancements in MLOps, machine learning, and cloud technologies.

Requirements

  • Bachelor's or Master's degree in Computer Science, Data Science, Engineering, or a related field.
  • 3+ years of experience in MLOps, DevOps, or related roles.
  • Strong understanding of machine learning lifecycle and best practices for model deployment.
  • Proficiency in programming languages such as Python and experience with ML frameworks like TensorFlow, PyTorch, and Scikit-learn.
  • Experience with CI/CD tools like Jenkins, GitLab CI, or CircleCI.
  • Proficiency in containerization (Docker) and orchestration (Kubernetes).
  • Strong problem-solving skills and attention to detail.
  • Experience with cloud platforms (AWS, Google Cloud, Azure).
  • Excellent communication and collaboration skills.

Preferred Qualifications

  • Experience with infrastructure as code tools like Terraform or Ansible.
  • Knowledge of data engineering and data pipeline tools (Apache Airflow, Kafka).
  • Familiarity with monitoring tools (Prometheus, Grafana) and logging frameworks (ELK stack).
  • Understanding of security best practices in MLOps.
